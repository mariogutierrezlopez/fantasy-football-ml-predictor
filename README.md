## üöÄ Key Features

- **Advanced GUI Architecture:** Developed with **PySide6** (Qt for Python), featuring a responsive layout and custom widgets.
- **ML Integration:** Seamless integration of **Scikit-learn** models (KNN, Random Forest, Decision Trees) within a desktop environment.
- **Data Visualization:** Embedded **Matplotlib** figures to visualize regression lines and clusters directly in the app.
- **Interactive Training:** Users can retrain models via the GUI using updated datasets.
